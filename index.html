🧲 마그넷 전략

VWMA 평균회귀

자석이 쇠를 끌어당기듯, 가격은 VWMA로 회귀합니다. 과도한 이격 시 진입, VWMA 터치 시 청산합니다.

  • ✓ VWMA에서 벌어지면 자동 진입
  • ✓ VWMA 터치 시 자동 청산
  • ✓ 하드 손절 1.5% 안전장치
  • ✓ 1시간봉 추세 필터

⚡ 얼티밋 전략

모멘텀 + 변동성

RSI, MACD, ADX, 피벗 포인트, 가격 액션 등 다중 지표를 종합 분석하여 강한 시그널에서만 진입합니다.

  • ✓ 4가지 독립 시그널
  • ✓ 볼륨 + ADX 이중 필터
  • ✓ 트레일링 스탑 수익 극대화
  • ✓ MTF 다중 타임프레임 필터
